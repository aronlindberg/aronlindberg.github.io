"The question is no longer whether AI belongs in the design process. It's whether the design process can remain human-centered once it does."

Human-centered design was built for a world where the designer was human and the machine was a tool. That world no longer exists.

This book rebuilds the framework for a world where AI can discover, frame, generate, build, and validate — not always as well as a skilled designer, but differently, faster, and sometimes better.

The five stages of the design process survive. But each one now requires new discipline: knowing when to hand off to AI, when to pull the work back, and how to stay accountable for everything the system produces.
